###Caça ao Tesouro

um mini jogo em **python**, criado pra praticar listas bidimensionais, loops e condicionais.  
a ideia é simples: achar o tesouro escondido no tabuleiro 5x5 antes que acabem as tentativas.

---

## objetivo
descobrir onde está o tesouro em até **7 tentativas**.  
a cada jogada, o jogador escolhe **linha** e **coluna**, e recebe **dicas** de direção (cima, baixo, esquerda, direita) até encontrar o tesouro.

---

## como jogar
1. garante que o **python 3** tá instalado.  
2. executa o arquivo no terminal:
   ```bash
   python caca_tesouro.py
